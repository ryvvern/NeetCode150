# NeetCode 150

This repository contains my solutions to the **NeetCode 150** problem set.  
The goal of this repository is to practice data structures and algorithms in a structured and consistent way.

## About

NeetCode 150 is a curated set of algorithm and data structure problems commonly used for technical interview preparation.  
This repository tracks my progress as I solve these problems.

## Structure

Problems are organized by topic.  
Each solution is written clearly with a psuedocode followed by the JavaScript code.
