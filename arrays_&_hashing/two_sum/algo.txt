Algorithm:

1. Create an empty hash map to store numbers we've seen, with each
   number's index as it's value

2. Loop throught the array with an index

3. For each number calculate it's complement

4. Check if the complement exists in the hash map
   a. If yes --> we have found the pair, return True
   b. If no --> add the current number and it's index to the hash
      map and move on

5. Repeat until the pair is found
